# 💰 Python Expense Tracker

A command-line based expense management system built using Python.  
This application allows users to record income and expenses, store financial data in files, and generate balance summaries.

---

## 📌 Project Overview

The **Python Expense Tracker** is designed to help users manage their personal finances directly from the command line.  

It enables users to:
- Add income entries
- Add expense entries
- Categorize transactions
- View total income and expenses
- Calculate remaining balance
- Store data persistently using file handling

This project demonstrates structured programming, file operations, and real-world problem solving using Python.

---

## 🚀 Features

✔ Add income transactions  
✔ Add expense transactions  
✔ Categorize expenses (Food, Rent, Transport, etc.)  
✔ Automatically calculate total balance  
✔ Save data to CSV/JSON file  
✔ Load saved data when program restarts  
✔ Simple and user-friendly CLI interface
